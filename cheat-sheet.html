Cheat Sheet

CSS

  • @import - Allow to import a style sheet into another style sheet
  • Font-Family - Specifies the font for an element
  • Margin - Used to create space around elements, outside of any defined borders
  • Padding - Used to generate space around an element's content, inside of any defined borders
  • Background-color - Sets the background color of an element
  • Color - Set the background color for HTML elements
  • Text-align - Used to set the horizontal alignment of a text
  • Vertical-align - Sets the vertical alignment of an element
  • Float - Specifies how an element should float
  • Font-size - Sets the size of a font
  • Text-shadow - Adds shadow to text
  • Display - Specifies the display behavior (the type of rendering box) of an element
  • Align-items - Specifies the default alignment for items inside a flexbox or grid container
  • Margin-left CSS - Sets the margin area on the left side of an element
  • Font-weight - Sets how thick or thin characters in text should be displayed.
  • Justify-content - Aligns the flexible container's items when the items do not use all available space on the main-axis (horizontally)
  • Width - Sets an element's width
  • Height - Sets the height of an element
  • Flex-direction - Specifies the direction of the flexible items
  • Border-radius - Defines the radius of an element's corners
  • Box-shadow - Attaches one or more shadows to an element
  • Background-image - Sets one or more background images for an element
  • Background-size - Specifies the size of the background images.
  • Margin-bottom - Sets the bottom margin of an element
  • Transitions - Change property values smoothly, over a given duration
  • Cursor - Specifies the mouse cursor to be displayed when pointing over an element
